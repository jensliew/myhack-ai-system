# Assets

Place your images here. They'll be accessible at `/assets/filename.ext` in the app.

## Suggested images:
- `hero-bg.jpg` — Landing page hero background
- `logo.svg` — Nexora logo

## Usage in code:
```tsx
<img src="/assets/hero-bg.jpg" alt="..." />
```

Or with Next.js Image component:
```tsx
import Image from "next/image";
<Image src="/assets/hero-bg.jpg" alt="..." width={1920} height={1080} />
```
